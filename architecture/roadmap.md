# SmartVolt AI — Roadmap

## Phase 1 — Foundation (Current)
- Architecture documents created.
- Core principles fixed.
- Multi-layer system structure defined.
- External memory model established.

## Phase 2 — System Mapping
- Build a full system map (Visio + AutoCAD).
- Define interfaces between layers.
- Describe data flows and orchestration logic.
- Introduce safety boundaries.

## Phase 3 — Local AI Stack
- Deploy baseline models via Ollama.
- Integrate vLLM as the core runtime.
- Prepare compatibility layer for TGI.
- Set up n8n orchestration nodes.

## Phase 4 — Automation and Tools
- AutoCAD automation (LISP + Python).
- Visio automation (Office Scripts).
- SmartVolt engineering templates.
- Multimodal logic blocks.

## Phase 5 — Smart Home & Hospitality
- Automation scenarios.
- Device orchestration framework.
- Safety and reliability modules.
- Monitoring and reporting.

## Phase 6 — Final Integration
- Full local deployment.
- Scalable module system.
- Detailed documentation package.
- Release of the open architectural spec.
